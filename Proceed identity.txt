### **Design Principles**
- **Visual-First Communication**: Every visualization tells a story in seconds

- **Strategic Impact**: Each design element serves a decision-making purpose
- **Professional Aesthetics**: Reflects organizational sophistication
- **8px Spacing Grid**: Consistent spacing throughout designs
- **Gradient Effects**: Magenta (#9e1f63) to Coral (#e05e3d) gradients for dynamic elements

### **Brand Application Standards**
- **Healthcare Logistics** positioning
- **Executive-level** communication standards
- **Clean, modern aesthetic** with accessibility focus
- **Professional image** maintenance across all touchpoints
- **Visual continuity** and brand recognition emphasis

- **Gradients**: Linear gradients using brand colors for depth
- **Shadows**: Subtle box shadows using rgba(66, 64, 70, 0.15)
- **Border Radius**: 8-12px for containers, 4px for buttons
- **Hover Effects**: Smooth transitions (0.3s ease)
